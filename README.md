## ✨ Características

- 🎨 **Design Moderno**: Interface elegante com animações suaves e gradientes
- 📱 **Totalmente Responsivo**: Otimizado para todos os dispositivos
- ⚡ **Performance**: Construído com Next.js 13 e otimizações avançadas

- 🎯 **Interativo**: Seções dinâmicas e elementos interativos

## 🛠️ Tecnologias Utilizadas

### Frontend
- **Next.js 13** - Framework React com App Router
- **React 18** - Biblioteca para interfaces de usuário
- **TypeScript** - Tipagem estática para JavaScript
- **TailwindCSS** - Framework CSS utilitário
- **Framer Motion** - Animações e transições

### Componentes e Estilos
- **Componentes Atômicos** - Arquitetura de design system
- **CSS Modules** - Estilos modulares

### Funcionalidades
- **Intersection Observer** - Animações baseadas em scroll
- **PDF Generation** - Geração de currículo em PDF
- **SEO Otimizado** - Meta tags e estrutura semântica

## 🚀 Como Executar o Projeto

### Pré-requisitos
- Node.js 18+ 
- Yarn ou npm

### Instalação

```bash
# Clone o repositório
git clone https://github.com/umrodrigues/curriculov2.git

# Entre no diretório
cd curriculov2

# Instale as dependências
yarn install
# ou
npm install
```

### Desenvolvimento

```bash
# Execute em modo de desenvolvimento
yarn dev
# ou
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000) no seu navegador.

### Produção

```bash
# Build para produção
yarn build
# ou
npm run build

# Execute em modo de produção
yarn start
# ou
npm start
```

## 🔧 Customização

### Cores
Edite as variáveis CSS em `src/styles/globals.css`:

```css
:root {
  --primary-orange: #f97316;
  --primary-blue: #3b82f6;
}
```

### Conteúdo
Atualize os dados em:
- `src/components/molecules/mocks/` - Dados dos componentes
- `src/components/molecules/` - Componentes de conteúdo
